## 中文
这是用于获取所有数据库（不是具体的db_owner权限或模式dbo）的数据库所有者的一些SQL逻辑，将指定的数据库所有者名称附加到数据库。 在创建数据库时，我也增加了人类可读的时间。


## English
Here’s some SQL logic to get the database owner across all databases ( not the specifically the db_owner right or schema dbo ), but the designated database owner name attached to the database. I’ve also taken the liberty of adding human readable time when the databases were created.

---
## Logic
```SQL
select
'database name' = db_name(database_id)
,   'owner name' = suser_name(owner_sid)
,   'database created'  = datename(dw, create_date) + ' ' + convert(char, create_date, 9) from
sys.databases
order by
db_name(database_id) asc


```
